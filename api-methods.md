# API methods summary
{: #api-methods}

{: #api-methods-runtime}

| Method | Description | Version | Reference |
|:-------|:------------|:-------:|:---------:|

| **Send user input to assistant (stateful)** | Send user input to an assistant and receive a response, with conversation state (including context data) stored by {{site.data.keyword.conversationshort}} during the session. | v2 | [cURL](/apidocs/assistant-v2#message){: external} [node](/apidocs/assistant-v2?code=node#message){: external} [Java](/apidocs/assistant-v2?code=java#message){: external} [python](/apidocs/assistant-v2?code=python#message){: external} |
